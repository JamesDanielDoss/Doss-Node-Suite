# Public Repo Structure Plan

## Purpose

`Doss-Node-Suite` is the public ComfyUI node repository for James Doss AI.

## Public Content

- Custom ComfyUI node code.
- Public examples.
- Public workflow snippets.
- Beginner-friendly install and usage docs.
- Changelog and release notes.

## Private Content That Does Not Belong Here

- Private workflow strategy.
- YouTube production plans.
- Raw recordings, exports, thumbnail drafts, or scripts.
- API keys, tokens, `.env` files, or account information.
- Full AI model files.

## Release Readiness Checklist

- Node has a simple `Doss_` name.
- Node appears in `docs/node_index.md`.
- README or docs explain the node in beginner language.
- Example workflow is public-safe.
- Changelog is updated.
- No large or private files are staged.
