# Examples

Example workflows and screenshots will live here once public nodes are released.

## Example Rules

- Keep workflows beginner-friendly.
- Include only the nodes needed to teach the concept.
- Do not include private production workflows.
- Do not include private prompts, paid assets, credentials, or full model files.

## Folder Use

- `examples/workflows/` stores small public workflow JSON examples.
- `examples/images/` stores small public example images when useful.
