# Install Guide

## Manual Install

1. Open your ComfyUI custom nodes folder:

```text
C:\AI\ComfyUI\custom_nodes
```

2. Clone this repository into that folder:

```powershell
git clone https://github.com/YOUR_ACCOUNT/Doss-Node-Suite.git
```

3. Restart ComfyUI.

## Update

```powershell
cd C:\AI\ComfyUI\custom_nodes\Doss-Node-Suite
git pull
```

## Troubleshooting

- Make sure the folder is inside `C:\AI\ComfyUI\custom_nodes`.
- Restart ComfyUI after installing or updating.
- If a node is missing, check the ComfyUI console for Python errors.
